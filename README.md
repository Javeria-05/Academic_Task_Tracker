# Academic Task Tracker System

A Python-based academic management system developed to help students organize and track their academic tasks efficiently.

---

## Project Description

The Academic Task Tracker System allows students to manage their academic activities including assignments, quizzes, mid exams, terminal exams, and project tasks. It provides an easy way to monitor task progress and maintain academic records.

---

## Key Features

✔ Add New Tasks  
✔ View All Tasks  
✔ Search Tasks  
✔ Update Task Status  
✔ Delete Tasks  
✔ Exit System
