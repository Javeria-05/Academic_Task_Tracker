## Overview

Academic Task Tracker is a command-line based software application developed to simplify academic task management for students. The system enables users to manage assignments, quizzes, examinations, and project deadlines through an interactive and structured workflow.

This project was developed as part of the Software Quality Engineering (SQE) course, focusing on software development, testing, bug reporting, code quality, security analysis, and version control.

---

## Core Functionalities

- Task Creation and Management
- Task Search and Retrieval
- Status Monitoring and Updates
- Task Deletion
- Academic Progress Tracking
- Interactive Menu-Based Navigation

---

## System Features

| Feature | Description |
|---------|-------------|
| Add Task | Create and save new academic tasks |
| View Tasks | Display all stored tasks |
| Search Task | Search tasks by course name |
| Update Status | Modify task completion progress |
| Delete Task | Remove completed or unwanted tasks |
| Exit | Safely terminate the application |
